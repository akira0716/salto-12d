# ビジネスルール

### 備品エンティティ

- 備品の状態は「利用可」「貸出中」「修理中」「廃棄済」のいずれかでなければならない
- 備品の状態は「修理中」から「貸出中」には遷移できない
- 備品の状態は「廃棄済」から他の状態には遷移できない

### 貸出申請エンティティ

- 利用終了希望日は利用開始希望日より後の日付でなければならない
- 貸出申請のステータスは「申請中」「承認済」「却下」のいずれかでなければならない

### 貸出エンティティ

- 返却予定日は貸出日より後の日付でなければならない
- 貸出のステータスは「貸出中」「返却済」のいずれかでなければならない
- 遅延判定：ステータスが「貸出中」であり、現在の日付が「返却予定日」を過ぎている場合、「遅延中」として扱う

### 利用者エンティティ

- 利用者に管理者権限があるかの判定

### 貸出ポリシーエンティティ

- 利用者が現在保持している「申請中」の貸出申請と「貸出中」の貸出の合計数は、最大3件を超えてはならない
- 利用者は遅延している貸出が1つでもある場合、新規申請ができない
- 1回の貸出期限は最大30日まで
