# 状態遷移図

## 1. 備品（EQUIPMENTS）のステータス状態遷移図

システムで管理される備品（個体）についての状態がどのように変わるかを示した図です。

```mermaid
stateDiagram-v2
    [*] --> 利用可 : 新規登録

    利用可 --> 貸出中 : 申請承認
    貸出中 --> 利用可 : 返却処理・受領完了

    利用可 --> 修理中 : 故障・不具合報告
    貸出中 --> 修理中 : 返却時の故障報告など
    修理中 --> 利用可 : 修理完了・復旧

    利用可 --> 廃棄済 : 経年劣化による処分
    修理中 --> 廃棄済 : 修理不能による処分
    貸出中 --> 廃棄済 : 紛失などによる未帰還

    廃棄済 --> [*]
```

## 2. 貸出申請（LOAN_REQUESTS）のステータス状態遷移図

利用者が申請してから、管理者が承認または却下するまでの状態遷移です。

```mermaid
stateDiagram-v2
    [*] --> 申請中 : 利用者が申請

    申請中 --> 承認済 : 管理者が承認
    申請中 --> 却下 : 管理者が却下（理由あり）

    承認済 --> [*]
    却下 --> [*]
```

## 3. 貸出（LOANS）のステータス状態遷移図

申請が「承認済」となった後に生成され、備品が貸し出されてから返却されるまでの状態遷移です。
※遅延の判定は、「貸出中」かつ現在日付が「返却予定日」を過ぎているかという計算で求められます。

```mermaid
stateDiagram-v2
    [*] --> 貸出中 : 管理者が承認し貸出開始

    貸出中 --> 返却済 : 管理者が返却処理・受領完了

    返却済 --> [*]
```
