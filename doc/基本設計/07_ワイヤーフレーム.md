# 画面設計（ワイヤーフレーム）

---

## 1. ログイン画面

利用者がシステムにアクセスする際の最初の画面です。

```text
+-------------------------------------------------+
|                                                 |
|            社内備品貸出管理システム             |
|                                                 |
|        [   ユーザーID (Email等)    ]            |
|        [   パスワード              ]            |
|                                                 |
|               [ ログイン ]                      |
|                                                 |
+-------------------------------------------------+
```

---

## 2. 備品一覧・検索画面（一般社員・管理者共通）

現在「利用可（在庫あり）」の備品を探す画面です。

```text
+-------------------------------------------------+
| [システム名]         [マイページ] [管理者メニュー] [ログアウト] |
+-------------------------------------------------+
| 備品を探す                                      |
|  [ キーワード検索... ] [ カテゴリ: PC ▼ ] [検索]|
+-------------------------------------------------+
| 検索結果 (利用可の個体のみ表示)                 |
| +---------------------------------------------+ |
| | [写真] ノートPC A (個体ID: 1001)           | |
| |        カテゴリ: PC本体                     | |
| |        ステータス: 利用可                   | |
| |        [ 詳細・貸出申請へ ]                 | |
| +---------------------------------------------+ |
| | [写真] モニター B (個体ID: 2005)           | |
| |        カテゴリ: モニター                   | |
| |        ステータス: 利用可                   | |
| |        [ 詳細・貸出申請へ ]                 | |
| +---------------------------------------------+ |
+-------------------------------------------------+
```

---

## 3. 貸出申請画面（一般社員用）

対象の備品の利用期間と目的を入力して申請する画面です。

```text
+-------------------------------------------------+
| [システム名]                          [ログアウト] |
+-------------------------------------------------+
| 貸出申請                                        |
| ----------------------------------------------- |
| 対象備品: ノートPC A (個体ID: 1001)             |
| カテゴリ: PC本体                                  |
| 状態: 利用可                                    |
|                                                 |
| ■ 利用期間を設定してください                      |
|   開始日: [ 2026/04/05  ]  ~  終了日: [ 2026/04/10 ]|
|                                                 |
| ■ 利用目的を記載してください（必須）              |
|  +-------------------------------------------+  |
|  | 出張時のプレゼン用として使用するため。    |  |
|  +-------------------------------------------+  |
|                                                 |
|       [ 戻る ]            [ 申請する ]          |
+-------------------------------------------------+
```

---

## 4. マイページ / 貸出状況確認（一般社員用）

自身の現在の貸出状況、申請状態、過去の履歴と「管理者の却下理由」等を確認できます。

```text
+-------------------------------------------------+
| [システム名]  [備品検索へ]            [ログアウト] |
+-------------------------------------------------+
| マイページ (貸出状況)                            |
| ----------------------------------------------- |
| ■ 現在の貸出・申請状況                          |
| | 備品名       | 期間        | ステータス | キャンセル |
| | ノートPC A   | 4/5~4/10    | 申請中    | [申請取消] |
| | プロジェクタ | 4/1~4/3     | 貸出中    |            |
|                                                 |
| ■ 過去の履歴                                    |
| | 備品名       | 期間        | ステータス | 備考       |
| | マウス x1    | 2/1~2/2     | 却下(*)   |            |
| | モニター24型 | 1/1~1/15    | 返却済    |            |
|                                                 |
|  (*) 却下理由: 貸出直前の点検で故障が発覚したため |
+-------------------------------------------------+
```

## 5. 管理者ダッシュボード（申請管理 / 承認・却下）

管理者が届いた申請を精査し、承認または理由付きで却下する画面です。

```text
+-------------------------------------------------+
| [管理者ダッシュボード]   [一般画面へ] [ログアウト] |
| ----------------------------------------------- |
| [ 申請管理 ] [ 貸出状況一覧 ] [ 備品マスタ管理 ]|
+-------------------------------------------------+
| ■ 貸出申請一覧 (未処理)                         |
|                                                 |
| 申請者: 山田太郎                                |
| 備品: ノートPC (個体ID: 1001)                   |
| 期間: 4/5 ~ 4/10                                |
| 目的: 出張利用                                  |
| 却下理由(却下する場合のみ入力): [              ]|
|     [ 承認する ]        [ 却下する ]            |
| ----------------------------------------------- |
| 申請者: 佐藤花子                                |
| 備品: プロジェクタ (個体ID: 4012)               |
| 期間: 4/15 ~ 4/15                               |
| 目的: 自宅での映画鑑賞                          |
| 却下理由(却下する場合のみ入力): [ 私物化のため ]|
|     [ 承認する ]        [ 却下する ]            |
+-------------------------------------------------+
```

## 6. 管理者ダッシュボード（返却受領・全体状況）

全社の貸出状況を確認し、返却遅延のチェックや物理的な返却の受領処理（ステータス更新）を行います。

```text
+-------------------------------------------------+
| [管理者ダッシュボード]   [一般画面へ] [ログアウト] |
| ----------------------------------------------- |
| [ 申請管理 ] [ 貸出状況一覧 ] [ 備品マスタ管理 ]|
+-------------------------------------------------+
| ■ 全社貸出状況・返却受領                        |
|   [ 絞り込み: 全て / 貸出中 ▼ ] [ 遅延のみ表示□ ]|
|                                                 |
| | 利用者 | 備品(ID)   | 終了予定日 | ステータス| 操作      |
| | 田中   | WiFi(500)  | 3/31(遅延!)| 貸出中   | [返却受領]|
| | 鈴木   | マウス(20) | 4/05       | 貸出中   | [返却受領]|
| | 佐藤   | PC(1005)   | 4/10       | 承認済   |           |
|                                                 |
| ※[返却受領]で「返却済」とし、個体を「利用可」へ |
+-------------------------------------------------+
```

---

## 7. 管理者ダッシュボード（備品マスタ・個体管理）

備品の追加や、故障等の個体ステータス変更を行う画面です。

```text
+-------------------------------------------------+
| [管理者ダッシュボード]   [一般画面へ] [ログアウト] |
| ----------------------------------------------- |
| [ 申請管理 ] [ 貸出状況一覧 ] [ 備品マスタ管理 ]|
+-------------------------------------------------+
| ■ 備品マスタ (個体管理)       [＋新規備品登録]  |
|   [ 備品分類（カテゴリ）管理画面へ ]            |
|                                                 |
| | ID  | 備品名       | カテゴリ | 個体ステータス | 操作   |
| | 1001| ノートPC A   | PC本体   | 利用可         | [編集] |
| | 1002| ノートPC A   | PC本体   | 貸出中         | [編集] |
| | 1003| ノートPC B   | PC本体   | 修理中         | [編集] |
| | 2005| モニター X   | モニター | 廃棄済         | [編集] |
|                                                 |
| ※[編集]から、ステータスの強制変更、名称変更などが可能 |
+-------------------------------------------------+
```
